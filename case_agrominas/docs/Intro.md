# Guia Regenerativo - Agrominas

## Biblioteca Digital por WhatsApp para pequenos e médios produtores

O projeto **Guia Regenerativo** surge como uma ponte tecnológica entre a ciência do solo e o cotidiano do produtor rural. Através de uma interface familiar (WhatsApp), democratizamos o acesso a dados técnicos sobre insumos regenerativos.

### Sobre a Empresa Parceira: **Agrominas Fertilizantes**

A **Agrominas Fertilizantes** é uma empresa atuante no segmento de fertilizantes e insumos minerais para o agronegócio, com foco em extração, produção e apoio à nutrição do solo. Fundada em 2023, a empresa opera como parceira estratégica no fornecimento de corretivos e no fomento de práticas sustentáveis que elevam a produtividade no campo, com forte presença em Minas Gerais e sede em São Paulo.

### Objetivo do Projeto

O objetivo central é desenvolver uma **biblioteca digital de insumos regenerativos** que armazene evidências técnicas de uso e resultados. Esta base de conhecimento é integrada a um **assistente inteligente via WhatsApp**, capaz de guiar produtores de pequeno e médio porte na consulta de artigos, práticas recomendadas e formas de aplicação, facilitando a transição para uma agricultura mais sustentável e eficiente.

### Sobre a Solução

A solução é composta por um ecossistema integrado que utiliza **Inteligência Artificial** e processamento de linguagem natural:

- **Interface Conversacional:** Um bot de WhatsApp que recebe perguntas em texto e documentos (PDFs de análise de solo).
- **Núcleo de IA (RAG):** Utilização da API da OpenAI com a técnica de _Retrieval-Augmented Generation_, garantindo que as respostas da IA sejam baseadas estritamente na base de dados técnica da Agrominas.
- **Painel Administrativo:** Uma interface web para curadoria de conteúdo, onde especialistas podem cadastrar, editar e gerenciar os artigos e evidências científicas.
- **Backend Robusto:** API desenvolvida para integrar a extração de dados de PDFs, o banco de dados Supabase e o fluxo de mensagens.
